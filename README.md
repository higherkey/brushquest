# 🪥 BrushQuest

A premium, web-based toothbrushing assistant for kids. Built for performance, visibility, and fun.

## ✨ Features
- **Precision Timing**: Drift-free countdown logic using the system clock.
- **High-Visibility UI**: Large-scale interface designed for bathroom use.
- **Interactive Character**: Rive-powered animations that react to your progress.
- **PWA Ready**: Works offline and can be installed on home screens.
- **Zero-Latency Audio**: Synthesized audio cues via the Web Audio API.
